LED Christmas Tree Guide

How to use your 3D‑printed lighted tree

Overview

This page explains how to operate your LED Christmas tree, adjust brightness, and switch between lighting modes.

Controls

Brightness Button

Press to cycle through brightness levels (Low → Medium → High → Max → Off).

Mode Button

Press to switch between lighting effects (Solid → Fade → Twinkle → Sparkle → Color Cycle).

Power

The tree powers on automatically when plugged in. It remembers the last mode and brightness used.
